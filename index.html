JavaScript is the programming language of HTML and the Web. Programming makes computers do what you want them to do.



  • JavaScript was initially created to “make web pages alive”.
  • The programs in this language are called scripts. They can be written right in a web page’s HTML and run automatically as the page loads.
  • Learned about Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run.

JavaScript Can Change HTML Content



One of many JavaScript HTML methods is getElementById().

This example uses the method to "find" an HTML element (with id="demo") and changes the element content (innerHTML) to "Hello JavaScript":

Example :
document.getElementById("demo").innerHTML = "Hello JavaScript";
